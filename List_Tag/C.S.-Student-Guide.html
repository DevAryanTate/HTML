Key Computer Science Terms

Algorithm
A step-by-step procedure to solve a problem or perform a computation.
Database
An organised collection of structured information or data, typical stored electronically.
Compiler
a program that translate source code written in one programming language into another language.
API
Application Programming interface; a set of functions and protocols for building software application.
